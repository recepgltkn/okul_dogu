# Block Grid Runner

Basit bir blok-temelli kodlama uygulaması örneği.

Özellikler:
- Blockly blokları (İleri, Sola Dön, Sağa Dön, Döngü)
- Grid üzerinde karakter yürütme ve adım adım oynatma
- Engeller ve yıldız hedefi; yıldız bulununca sonraki seviye kilidi açılır
- Seviye ilerlemesi `localStorage` ile saklanır

Çalıştırma:
1. Bu dizindeki `index.html` dosyasını bir tarayıcıda açın.
2. Sol taraftan blokları sürükleyip birleştirin.
3. `Çalıştır` veya `Adım` ile programı test edin.

Gelecek iyileştirmeler:
- Google login / kullanıcı kaydı
- Daha fazla blok, hata geri bildirimleri, sesler
- Mobil uyum ve daha fazla seviye editörü
